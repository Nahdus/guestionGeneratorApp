# Automatic Question Generator [AQG]

Automatic Question Generator from Text

  
  

Prerequisites

-------------

```

- Python 3.5+

- NLTK

- SpaCy

- NumPy

```

  

## Quickstart

  

### 1.  start REST service server

  

	1.1 pip install -r requirements.txt
                   
	1.2. python AutomaticQuestionGenerator\app.py

  

### 2. start react app

	 2.1 npm install
	 
	 2.2 npm start
